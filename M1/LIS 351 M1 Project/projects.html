Introduction

All of my personal projects on this page will be related to Machine learning. I am trying to learn more about this topic and project based learning is one of the best ways I learn.

Project List

  • Fantasy Premier League (FPL) ML model – This projects helps players of FPL build the best team by predicting results.
  • Amazon item algorithm – Suggests items you might like on amazon based on what you viewed in the past.
  • Fashion trend predictor – Predicts what trends will come up in the fashion industry.

Project Details

FPL model
Used the premier leagues database of stats, as well as the fpl database to predict what the best team would be every week. Also gave users the options to select individual players they like and calculated the best team possible based on the new parameters.
Amazon algorithm
Asked users questions on what they usually buy and outputted amazon links they might like based on their answers. Used existing Amazon database of items, and also added a way for users to control how much information is inputted the model.
Fashion trend predictor
Scrapes web for existing data on fashion trends, and use sentiment analysis to see current, and predict future trends. Added option for user to see if specific trends are currently big, or will be big in the future.
